Our Goal

  • To develop a competitive HTML based slideshow system that takes advantage of modern web technologies.
    • HTML5: More sophisticated tags
    • JQuery: Ease of use and development and cross-browser
    • CSS3 enhanced effects
  • Also have a level of integration with the web without depending on it.
    • Use of google API
    • Dynamic Tag Cloud Generation

Fully browser based 2 dimensional slideshow

Custom OWL Brush: Developed using regular expressions

<owl:Class>
  <owl:oneOf rdf:parseType="Collection">
    <owl:Thing rdf:about="#Eurasia"></owl:Thing>
    <owl:Thing rdf:about="#Africa"></owl:Thing>
    <owl:Thing rdf:about="#NorthAmerica"></owl:Thing>
    <owl:Thing rdf:about="#SouthAmerica"></owl:Thing>
    <owl:Thing rdf:about="#Australia"></owl:Thing>
    <owl:Thing rdf:about="#Antarctica"></owl:Thing>
  </owl:oneOf>
</owl:Class>

This is the regular expression used to parse this:

		  var	regexOne = new XRegExp('\\<*\\s*(? [^\\:?]+)\\:(? [^\\s\\>]+)\\/*\\>*', 'xg');
			var	regexTwo = new XRegExp('(? [^\\=?]+)\\=(? ".*?"|\'.*?\'|\\w+)\\/*\\>*', 'xg');
